## Projects

### [Amazon Web Scraper](https://github.com/Moharabie99/Amazon-Web-Scraper)

The Amazon Web Scraper is a data extraction tool designed to gather information about products listed on the Amazon website. Leveraging web scraping techniques, the tool collects essential details such as product titles and prices, providing valuable insights for price tracking and market analysis.

### [Superstore Data Analysis](https://github.com/Moharabie99/Store-Numbers)

The Store Data Analysis Project is a comprehensive exploration of sales and performance data spanning four years (2014-2017) for a retail store. Through extensive data analysis and visualization, the project aims to uncover patterns, trends, and key insights that can inform strategic decision-making for both short-term and long-term planning.

### [Covid-19 Global Numbers EDA](https://github.com/Moharabie99/Data-exploration--Covid-19-global-numbers)

In the face of the global COVID-19 pandemic, I embarked on a data exploration journey to gain deeper insights into the dynamics of the virus's impact. Leveraging my analytical skills, I delved into extensive datasets, uncovering patterns, trends, and critical metrics that provided a comprehensive understanding of the pandemic's evolution.

### [MyAnimeList Data Exploration](https://github.com/Moharabie99/MAL-data-2023)

Embarking on a journey through the colorful world of anime, I ventured into the vast dataset extracted from MyAnimeList, covering the period from 1960 to 2023. This data-driven exploration aimed to unravel trends among users and unveil insights into the evolution of anime preferences over the years.

### [Nashville Housing Report](https://github.com/Moharabie99/Nashville-housing/tree/master)

This project involves cleaning and standardizing the Nashville housing dataset. The dataset contains information about property sales, including dates, addresses, and sale prices. The goal is to prepare the data for further analysis by resolving inconsistencies and improving data integrity.
